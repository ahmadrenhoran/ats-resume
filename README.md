# buildmyresume

Web for you to create resume with ATS format.

## Demo

[Demo Website](https://cvbuilder-ahmadrenhoran.netlify.app/)

### Stack
- Vue 3
- Tailwind v4
- Pinia
- html2pdf.js
- Vue Router
- Quill for Rich Text

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
